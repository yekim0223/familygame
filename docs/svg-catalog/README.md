# Family Quest SVG 에셋 카탈로그

> 전체 SVG 파일 목록과 경로 정리. 실제 파일은 `/public/assets/` 에 있습니다.

---

## 폴더 구조 개요

```
public/assets/
  characters/    ← 캐릭터 SVG (159종)
  pets/          ← 반려동물 SVG (50종)
  backgrounds/   ← 내 땅 배경 SVG (50종)
  gear/
    weapon/      ← 무기 SVG (10종)
    helmet/      ← 투구 SVG (5종)
    shield/      ← 방패 SVG (5종)
    armor/       ← 갑옷 SVG (5종)
  icons/         ← UI 아이콘 SVG (19종)
```
